# How to commit (non-team member)
If you are to commit to the repository, (and you are not a member of the team) fork it, and then you can do what you like to it and then sync, then push to the repository.

# How to commit (team member)
If you are a member of HHR, the computer at the club will have git already installed. From there, you can:
 1. Make edits to the repository (via Visual Studio Code)
 2. open git
 3. navigate to repository (should be on desktop - cd Desktop/robot-code)
 0. If someone has made changes to the repository, you can sync it via "git pull origin main"
 4. "git add ."
 5. "git commit -m "Your Message Here"
 6. "git push" (if that asks for credentials, type in the username & password, written on computer)
