# How to commit (non-team member)
If you are to commit to the repository, (and you are not a member of the team) fork it, and then you can do what you like to it and then sync, then push to the repository.

# How to commit (team member)
If you are a member of HHR, the computer at the club will have git already installed. From there, you can:
 1. open git
 2. navigate to repository (should be on desktop - cd Desktop/robot-code)
 3. "git pull origin main"
 4. Make edits to the repository (via Visual Studio Code)
 5. "git add ." - Adds your code to the github cloud
 6. "git commit -m "Your Message Here" " - Adds a message for the push request (DO NOT LEAVE THAT BLANK)
 7. "git push" (if that asks for credentials, type in the username & password, written on computer) - Adds it to the repository
